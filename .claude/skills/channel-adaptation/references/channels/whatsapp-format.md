# WhatsApp Format Guide

## Style: Casual & Rapid

Friendly, concise, mobile-native communication for WhatsApp channel.

## Required Structure

```
[Single paragraph response]
[1-2 relevant emojis]
```

## Critical Constraint

**MAXIMUM 60 WORDS (STRICT LIMIT)**

Every WhatsApp message MUST be under 60 words. No exceptions.

## Required Elements

### 1. No Formal Greeting
- Skip "Hello [Name]," unless it's the first message of the day
- Jump straight to the solution
- Exception: First message of day → "Good morning! [response] 👋"

### 2. Single Paragraph
- Keep everything in one paragraph
- Use line breaks sparingly (max 2-3 lines)
- No numbered lists (too formal)

### 3. Emojis (1-2 relevant)
Approved emojis:
- 🚀 Launch/speed/quick action
- ✅ Confirmation/success
- 🛠️ Fix/technical work
- 👋 Greeting
- 📊 Data/reports
- 💡 Idea/tip
- ⚡ Urgent/fast
- 🔗 Link/connection

### 4. Bold for Key Terms
Use WhatsApp markdown: `*important term*`
- *Settings*
- *Regenerate Token*
- *Pro feature*

## Formatting Guidelines

### WhatsApp Markdown
- **Bold**: `*text*` → *text*
- **Italic**: `_text_` → _text_ (use sparingly)
- **Strikethrough**: `~text~` → ~text~ (rarely used)

### Links
- Provide links for detailed instructions
- Use short, descriptive text: "Upgrade here: [link]"
- Don't embed long URLs in the message

### Word Count Strategy
To stay under 60 words:
1. Remove all filler words
2. Use contractions (you'll, it'll, we've)
3. Provide link instead of detailed steps
4. Focus on immediate action

## Complete Examples

### Example 1: Technical Issue Resolution

**Raw Response**:
"The Slack integration isn't syncing because the OAuth token expired. You need to reconnect your Slack workspace from the integrations page. Go to Settings > Integrations, find Slack, and click Reconnect Workspace."

**WhatsApp Format**:
```
Your Slack token expired! Quick fix: *Settings > Integrations* → find Slack → hit *Reconnect*. Syncs in 2-3 mins after that 🛠️ Need help?
```
(26 words ✅)

### Example 2: Feature Request

**Raw Response**:
"The Gantt chart feature is only available on Professional and Enterprise plans. You're currently on the Starter plan. You can upgrade to access this feature. Once upgraded, it will appear in your Project Timeline view."

**WhatsApp Format**:
```
Your Gantt chart is a *Pro feature*. Upgrade here: [link] and it'll show up in Timeline > Gantt View. Takes 2 mins to set up! 🚀 Let me know if you need help.
```
(35 words ✅)

### Example 3: Urgent Issue with Empathy

**Raw Response**:
"Your project data is safe. The export failed due to a temporary server issue. I've manually triggered the export and it should complete in 5 minutes. You'll receive an email with the download link."

**WhatsApp Format**:
```
Your data's safe! Export failed due to server glitch (now fixed). I've triggered it manually—you'll get the download link via email in ~5 mins ⚡ Let me know if it doesn't arrive.
```
(35 words ✅)

### Example 4: Multiple Options

**Raw Response**:
"You have two options: 1) Upgrade to Professional plan for $49/month to access Gantt charts, or 2) Use the Timeline view which is included in your current plan and provides similar functionality."

**WhatsApp Format**:
```
Two options: upgrade to *Pro* ($49/mo) for Gantt charts [link], or use *Timeline view* (free on your plan) for similar features. Which works better for you?
```
(28 words ✅)

### Example 5: First Message of Day

**Raw Response**:
"I've resolved your API authentication issue. The token has been regenerated and your webhook is now working correctly."

**WhatsApp Format**:
```
Good morning! Your API issue is fixed—token regenerated and webhook's working now ✅ Let me know if you see any more errors.
```
(23 words ✅)

## Edge Cases

### Long Technical Explanations
**Don't**: Try to fit everything in 60 words
**Do**: Summarize + provide link
```
Quick fix: *Settings > API Keys* → Regenerate. Full guide here: [link] 🛠️ Takes 2 mins. Need help walking through it?
```

### Multiple Issues in One Message
**Don't**: Address all issues in one message
**Do**: Prioritize most urgent, mention others
```
Fixed your Slack sync (token issue) ✅ For the export problem, I'll send a separate message with details. Which is more urgent for you right now?
```

### Customer Used Technical Terms
**Do**: Mirror their language level
```
Customer: "My API endpoint is returning 401s"
Response: "Your API token expired! Regenerate in *Settings > API Keys* and update your endpoint config 🛠️"
```

### Time-Sensitive/Urgent Issues
Lead with urgency acknowledgment:
```
Quick fix for you: [solution] ⚡ Should work immediately. Let me know if it doesn't!
```

## Tone Adjustments by Situation

| Situation | WhatsApp Approach |
|-----------|------------------|
| **Urgent/Blocking** | Lead with "Quick fix:" or use ⚡ emoji |
| **Data Loss Concern** | Start with "Your data's safe!" |
| **Confusion** | Use simple language, offer to walk through |
| **Frustration** | Acknowledge briefly, focus on solution |
| **Feature Request** | Present options concisely with links |

## Common Mistakes to Avoid

❌ **Too formal**: "Hello Sarah, I hope this message finds you well..."
✅ **Correct**: "Quick fix for your Slack sync..."

❌ **Too long**: Trying to explain everything in detail
✅ **Correct**: Summarize + provide link for details

❌ **No emojis**: Plain text feels cold on WhatsApp
✅ **Correct**: Add 1-2 relevant emojis

❌ **Multiple paragraphs**: Breaks mobile reading flow
✅ **Correct**: Single paragraph, max 2-3 line breaks

❌ **Numbered lists**: Too formal for WhatsApp
✅ **Correct**: Use arrows (→) or brief descriptions

- [ ] Single paragraph format
- [ ] 1-2 relevant emojis included
- [ ] No formal greeting (unless first message of day)
