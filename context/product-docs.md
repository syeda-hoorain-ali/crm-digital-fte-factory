# CloudStream Product Documentation: The Knowledge Base

## 1. Core Modules

### **SmartBoards (Project Management)**
* **Kanban View:** Default view for tasks. Supports custom columns.
* **Gantt Chart:** Available only on **Pro** and **Enterprise** plans.
* **Automation:** Tasks can be set to automatically move to "Archive" after 30 days of inactivity.

### **TimeStream (Time Tracking)**
* **Chrome Extension:** Allows one-click tracking from any browser tab.
* **Manual Entry:** Users can add time retroactively via the "Timesheet" tab.
* **Billable vs. Non-Billable:** A toggle on every task to determine if the time is pushed to the final invoice.

### **ClientBridge (The Portal)**
* **White-Labeling:** Agencies can add their own logo and brand colors (Pro Plan only).
* **Approvals:** Clients can leave "Sticky Note" comments on design files (PDF, PNG, JPG).

## 3. Common Troubleshooting (L1 Support)

### **Login Issues**
* **Password Reset:** Users should click "Forgot Password" on the login screen. If the email doesn't arrive, check the Spam folder or verify if the account was created via Google Auth (SSO).
* **Two-Factor Authentication (2FA):** If a user loses their 2FA device, they must provide their "Recovery Code" generated during setup. If lost, escalate to human support.

### **Integrations**
* **Stripe:** To sync invoices, go to Settings > Billing > Connect Stripe. If "Connection Failed" appears, ensure the user has "Administrator" permissions in Stripe.
* **Slack:** Provides notifications when a client approves a design. Configuration is found under the "Integrations" tab.

### **Billing & Invoices**
* **Updating Credit Card:** Settings > Billing > Payment Methods.
* **Refund Policy:** CloudStream offers a 14-day "No Questions Asked" refund for new subscriptions. Beyond 14 days, refunds are handled on a case-by-case basis by human agents.
